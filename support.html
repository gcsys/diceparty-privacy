Questions fréquentes

Comment créer mes propres règles ?
Depuis le menu, ouvre « Règles », choisis le mode (1 ou 2 dés), saisis ta règle et appuie sur « + Ajouter ». Tes règles sont enregistrées sur ton appareil.

Comment supprimer les publicités ?
Dans les réglages (roue dentée), utilise « Supprimer les pubs ». L'achat se restaure via « Restaurer mes achats » sur tes autres appareils.

Mes règles ont disparu ?
Les règles personnalisées sont stockées localement ; les réinstaller ou effacer les données de l'app les supprime.
